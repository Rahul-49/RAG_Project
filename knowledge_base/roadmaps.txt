ON-CAMPUS PLACEMENT PREPARATION ROADMAPS
SERVICE-BASED IT COMPANIES (INDIA)

This document contains structured preparation roadmaps for major service-based IT companies hiring through on-campus drives. Each roadmap includes aptitude, coding, computer science fundamentals, and interview preparation focus areas. The content is optimized for Retrieval-Augmented Generation (RAG) systems.

==================================================================

COMPANY: :contentReference[oaicite:0]{index=0} (TCS)

Roadmap Focus:
Aptitude-heavy with gradual increase in coding difficulty depending on role (Ninja, Digital, Prime).

Preparation Areas:

Aptitude:
- Quantitative aptitude (percentages, ratios, time & work, time-speed-distance)
- Logical reasoning (arrangements, puzzles, data sufficiency)
- Verbal ability (reading comprehension, grammar, vocabulary)

Coding:
- Programming fundamentals (C / Java / Python)
- Arrays and strings
- Basic data structures
- For Digital/Prime: OOPS, intermediate DSA, time complexity

Computer Science Fundamentals:
- DBMS basics and SQL queries
- Operating systems basics
- Software engineering fundamentals

Interview Preparation:
- Simple coding explanation
- Project explanation
- Managerial and HR scenario questions
