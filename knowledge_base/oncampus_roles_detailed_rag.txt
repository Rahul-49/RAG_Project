COMPANY: Tata Consultancy Services (TCS)
------------------------------------------------------------------

Role ID: tcs_role_001
Role Name: Ninja (Assistant System Engineer – Ninja)
Pay Scale: 3.3–3.6 LPA

Role Overview:
Entry-level engineering role focused on application support, development, and testing with foundational programming requirements.

Required Skills:
- Programming fundamentals (C, Java, Python)
- Basic Data Structures (arrays, strings)
- SQL and DBMS fundamentals
- Operating Systems basics
- Strong aptitude and communication skills

Placement Rounds:
1. Online Aptitude Test
   - Quantitative aptitude
   - Logical reasoning
   - Verbal ability
2. Technical Interview
   - Programming basics
   - Simple coding problems
   - DBMS and OS questions
3. Managerial Interview
   - Situation-based questions
   - Teamwork and adaptability
4. HR Interview
   - Company fit
   - Relocation and policy discussion
